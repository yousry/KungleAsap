What it is:

Kungles "Completely Automated Public Turing test to tell Computers and Humans Apart" is a web service, designed to protect your webpage from malicious bots and crawler. In contrast to other systems, Kungle Captcha uses text questions instead of pictures for the test and can therefore be uses by disabled persons.

How does it work:

A Random Gap Text is generated. In each sentence the search word is removed. You have to guess the missing word. An optional hint list can be activated after a period of time. Each test is unique and removed from the catalogue once the correct answer was given.

Technology:

The Service is capable to generate more than 1.000.000 unique questions per month. KungleImp.de has a quota of 50.000 questions.
